# TVM Error Codes



`0`	TVM terminated successfully

`1`	TVM terminated successfully:  alternative code

`2`	Stack underflow

`3`	Stack overflow

`4`	Integer overflow

`5`	Range check error

`6`	Invalid opcode

`7`	Type check error

`8`	Cell overflow

`9`	Cell underflow

`10` Dictionary error

`11` Unknown error

`12` Fatal error

`13` Out of gas: the contract is either low on gas, or its limit is exceeded
